# 🟪 OkakPix — Pixel Battle

Мультиплеєрна піксель-баттл гра. Малюйте разом з друзями!

## ✨ Особливості
- 🎨 Реальний мультиплеєр через WebSocket
- 🖊 Два інструменти: Піксель і Олівець
- 📊 Стак 120 пікселів, +1/сек
- 👑 Акаунт admin без кулдауну
- 🔊 Звуки при малюванні
- 🗺️ Полотно 1000×1000 з earth.png
- 💾 Автозбереження кожні 30 секунд
- 📱 Підтримка тачскрінів та мобільних

---

## 🚀 Запуск локально

### 1. Встановити Node.js
https://nodejs.org (версія 16+)

### 2. Встановити залежності
```bash
npm install
```

### 3. Запустити сервер
```bash
npm start
```

### 4. Відкрити у браузері
```
http://localhost:3000
```

## 🌍 Хостинг для друзів

### Варіант A — Railway (безкоштовно, рекомендовано)

1. Зареєструйтесь на https://railway.app
2. Натисніть "New Project" → "Deploy from GitHub"
3. Завантажте папку `okakpix` на GitHub
4. Railway автоматично виявить Node.js і задеплоїть
5. Отримаєте посилання типу `https://okakpix-xxx.railway.app`
6. Поділіться з друзями!

### Варіант B — Render (безкоштовно)

1. Зареєструйтесь на https://render.com
2. "New Web Service" → підключіть GitHub репо
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Готово!

### Варіант C — VPS (Україна/ЄС)

```bash
# На сервері:
git clone <ваш-репо>
cd okakpix
npm install
# Встановити PM2 для фонового запуску
npm install -g pm2
pm2 start server.js --name okakpix
pm2 save
pm2 startup
```

Порт: 3000. Налаштуйте Nginx як reverse proxy якщо потрібен домен.

### Варіант D — ngrok (швидкий тест)

```bash
npm start
# В іншому терміналі:
npx ngrok http 3000
# Отримаєте посилання — дайте друзям!
```

## ⌨️ Керування

| Дія | Дія |
|-----|-----|
| ЛКМ | Поставити піксель |
| ПКМ / Середня кнопка | Панорамування |
| Пробіл + ЛКМ | Панорамування |
| Колесо миші | Зум |
| `1` | Інструмент Піксель |
| `2` | Інструмент Олівець |
| `+` / `-` | Зум |

**На телефоні:**
- Тап = піксель
- Один палець + олівець = малювання
- Два пальці = зум і панорамування
